# 📝 Student Feedback Sentiment Analysis

### Extracting Actionable Insights from Student Feedback Using Data Analytics & NLP

Student Feedback Sentiment Analysis is a Natural Language Processing (NLP) and Data Analytics project focused on analyzing student feedback collected from college events to understand satisfaction levels, identify recurring themes, and generate actionable recommendations.

The project combines structured rating analysis with sentiment analysis techniques to transform raw feedback into meaningful insights that can support event planning and student engagement initiatives.

---

## 📌 Project Overview

Feedback plays a crucial role in evaluating the success of events and understanding participant experiences.

This project analyzes student feedback data collected through surveys to uncover patterns in satisfaction, identify strengths and weaknesses, and measure overall sentiment using Natural Language Processing techniques.

The objective is to convert unstructured feedback into actionable insights that can help organizers improve future events.

### Objectives

* Analyze student satisfaction levels
* Evaluate event performance
* Perform sentiment analysis on textual feedback
* Identify frequently mentioned topics and keywords
* Generate data-driven recommendations
* Visualize insights through charts and word clouds
